SixPrizer beta is ready if anyone wants to help test it.

Link: [SixPrizer link]

It is a Pokemon TCG testing tracker that helps you log games, track deck versions, and spot patterns in matchups and results.

If you try it:

- sign up and check spam for the confirmation email
- create a deck and a test version
- log some real testing games if you can
- check Dashboard, Matchups, and Review after you have a few games in

Please post in the group or DM me if:

- something breaks
- mobile layout looks wrong
- something feels confusing or too slow
- a coaching insight feels unclear or wrong

Screenshots plus device/browser help a lot.

There is also a `Feedback` page inside the app if you want to save a bug or suggestion there directly.

It is still a beta, so rough edges are useful feedback.
